# Min-URL - Frontend

Min-URL es una aplicación web que permite a los usuarios acortar enlaces largos y generar un código QR asociado a cada URL acortada. Este es el repositorio del frontend, que se está desarrollando utilizando **React**, **Tailwind CSS** y **NextUI** para lograr una interfaz de usuario moderna, eficiente y responsive.

## Estado del proyecto

Actualmente, el proyecto está en la fase de **maquetación**, centrado en el diseño de la interfaz de usuario. El objetivo principal en esta etapa es asegurar una experiencia de usuario sencilla y funcional.

## Tecnologías utilizadas

- **React**: Biblioteca principal para construir la interfaz de usuario.
- **Tailwind CSS**: Framework de CSS que permite un desarrollo rápido con clases predefinidas, enfocándose en un diseño responsive y minimalista.
- **NextUI**: Biblioteca de componentes UI que proporciona estilos prediseñados y personalizables para React, facilitando la creación de una interfaz consistente y moderna.
